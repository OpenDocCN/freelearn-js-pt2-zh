# 第二部分：构建应用程序

在这个动手实践的环节，你将创建一个 Deno 应用程序，从服务器端渲染的网站开始，然后过渡到**代表性状态传输**（**REST**）**应用程序编程接口**（**APIs**），这些接口与数据库相连并具备认证功能。
